# To check the node version
```
node --version
```

# To initialise the firebase console
```
firebase init
```

# To start the server
```
npm run server
```

# To create new react project using npm & yarn
```
npm create-react-app folder_name 
yarn create react-app folder_name 
```

# To start the client
```
yarn start
```
